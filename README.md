Install
=======

Clone repo

    git clone https://github.com/bekbulatov/.home.git ~/.home

Install necessary modules

	cd ~/.home
	make install-bash
	make install-cheat
	make install-git
	make install-mysql
	make install-python
	make install-ruby
	make install-screen
	make install-tmux
	make install-vim

Reload `.bash_profile`

    source ~/.bash_profile
